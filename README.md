A custom Catan-inspired board game focused on sustainable energy! Build waste plants, upgrade to incineration plants, and be the first to reach 10 victory points.

## How to Play

1. Open `index.html` in your web browser
2. Select number of players (2-4)
3. Toggle robber mechanic if desired
4. Click "Start Game" and enjoy!

## Game Features

### Resources
- **Money** (green) - Replaces ore
- **Garbage** (gray) - Replaces wheat
- **Energy** (blue) - Replaces brick
- **License** (white) - Replaces wood

### Buildings
- **Waste Plants** - Worth 1 VP (replaces settlements)
- **Incineration Plants** - Worth 2 VP (replaces cities)

### Game Mechanics
- 2-4 player support
- Classic Catan 19-hex board layout
- 4 starting corners: Major, Builder, Investor, Supplier
- Dice rolling for resource distribution
- Development cards system
- Bank trading (4:1 ratio)
- Toggleable robber mechanic
- Largest army bonus (+2 VP)
- Victory condition: 10 points

## Technical Details

- Single HTML file implementation
- Vanilla JavaScript (no frameworks)
- Responsive design
- Complete game state management
- Pass-and-play local multiplayer
